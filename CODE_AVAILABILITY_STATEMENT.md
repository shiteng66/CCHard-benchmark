# Recommended Code Availability Statement

The custom code used to demonstrate the public CCHard evaluation workflow is released with the dataset under the MIT License. The release provides an OpenAI-compatible model-calling interface, resumable inference, explicit-rubric LLM-assisted scoring, deterministic bootstrap aggregation, figure generation, tests and a credential-free offline demonstration. The code version is tied to the dataset release tag and will be archived with the corresponding Zenodo record.

The public implementation supports the five case-derived subtasks contained in the released sample. It is a clean-room reference implementation of the reported workflow rather than a redistribution of internal historical scripts. The complete unpublished item bank, third-party source materials, private infrastructure configuration and credentials are not included. Consequently, the public package supports verification of the evaluation mechanics but does not independently reproduce the manuscript's full-benchmark numerical results.

After deposit, replace repository placeholders in the manuscript with the permanent GitHub release URL and Zenodo DOI.
