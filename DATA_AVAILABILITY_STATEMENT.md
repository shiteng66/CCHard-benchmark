# Recommended Data Availability Statement

An openly accessible, copyright-cleared and fully de-identified subset of CCHard is released with this study. The release contains 250 case-derived task items from five approved subtasks, task prompts, structured reference answers, scoring rubrics, a data dictionary, numeric summary statistics and runnable public evaluation code. The released subset contains no textbook questions, clinical-guideline source text, medical-terminology source records or professional-examination questions.

The count scopes are distinct. The approximately 130,000-item `full_constructed_pool` is the total task-formatted collection described in manuscript Section 2.2. A dynamically selected fixed-size `evaluation_working_subset` of 24,014 records is used for evaluation to control cost and reduce benchmark invalidation. The 250-item `public_release_sample` was sampled from that working subset only after item-level provenance, de-identification and copyright review. It supports verification of the benchmark schema, prompting, scoring, statistics and plotting workflow; it is not representative of, and cannot reproduce performance estimates for, the full benchmark.

The complete item bank is not publicly distributed because it contains third-party copyrighted source categories and additional case-derived materials that have not completed item-level public-release review. At deposit, cite the GitHub release tag and the DOI assigned by Zenodo. Replace any temporary manuscript project address with the permanent GitHub and Zenodo landing pages after upload.
